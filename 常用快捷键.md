# 常用快捷键

# IDEA快捷键

- `Ctrl+X` 删除行
- `Ctrl+D` 复制行
- `Ctrl+E` 最近打开的文件
- `Ctrl+H` 显示类结构图
- `Ctrl+N` 查找类
- `Alt+Enter` 导入包,自动修正
- `Ctrl+Alt+L` 格式化代码
- `Ctrl+/`  注释
- `Alt+上下键` 可以使代码上下行移动
- `双击shitf` 全局搜索

# HTML快速写法

***快捷键***

- `Alt+鼠标左键`，选中多行进行编辑，出现多个光标。
- `Ctrl+Shift+L`，选中与当前被选中行相同的内容进行编辑，文本末尾出现多个光标。
- 按住`Ctrl + Alt`，再按键盘上的上或下键，可以使一列上出现多个光标。

***快速写法***

- 输入`lorem`，然后按下Tab键，会自动生成文本内容。`lorem+数字`，可以生成一定长度的文本内容。
- `.row.align*3`  可以迅速写出
    
    ```jsx
    <div class="row align"></div>
    <div class="row align"></div>
    <div class="row align"></div>
    ```
    

# 搜索引擎语法

- `“搜索市场份额”`   精确搜索
- `intitle:` 搜索市场份额
- `intext:` 搜索市场份额
- 搜索市场份额 `inurl:video`
- 搜索市场份额 `site:zhihu.com`
- 搜索市场份额 `filetype:pdf`
